# Lab1
Quadratic Equation Solver

## Description
This program allows you to solve quadratic equations.

## Setup

### 1. Clone this repository:
```bash
 git clone
```

### 2. Start:
Go to the directory of project and run command:

Interactive mode
```bash
 node quaEquation.js
```
Noninteractive mode
```bash
 node quaEquationFile.js
 num.txt
```
